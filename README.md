## 功能

- 自动更新频道列表
- 多直播源
- 播放异常自动换源/刷新
- 应用自定义设置
- 适配手机、平板和电视

## 优势

- 直接使用官网直播链接，非常稳定可靠
- 兼容 m3u8 链接

## 局限

- 比起直接加载直播源，载入频道稍长，需要等待一定时间
- 使用的是网页播放器播放，对设备性能有一定的要求
- 无法选择视频清晰度

## FAQ

**Q: 在电视机/盒子上无法自动全屏？**
<br/>
A: 一般是设备 WebView 的问题，待X5内核下载完成，重启 App 即可。若无法自动下载X5内核，请在设置-TBS调试界面内手动安装。
<br/>
<br/>
